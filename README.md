# Profile Page + Posts Feed

## Технологии

### Frontend
- React + TypeScript
- TanStack Query
- Zustand
- Material UI
  
### Backend
- NestJS
- PostgreSQL + TypeORM
- JWT (access + refresh tokens)
- Multer (загрузка файлов)

### DevOps
- Docker
- docker compose

---


## Требования перед запуском

- Node.js >= 18
- npm
- Docker
- Docker Compose v2

## Запуск через Docker 

git clone git@github.com:MikhailSokol/profile-feed.git
cd profile-feed

docker compose up
